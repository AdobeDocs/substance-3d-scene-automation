class Image

Bases: object

Image object represents an image buffer from a file path or as rendering results.

Image must be created with the class method Image.from_filepath() Members: * height (int): image height, in pixels * width (int): image width, in pixels * channels (List[str, …]): image channel names

over(foreground, background, roi=None, force_fg_colorspace=None, force_bg_colorspace=None)
Compose foreground and background images with an over operation. Final Image dimension will be the larger

of the two images.

Parameters:

  • foreground (substance3d.api.image.image.Image | str | pathlib.Path) – a valid image filepath or Image object for the foreground

  • background (substance3d.api.image.image.Image | str | pathlib.Path) – a valid image filepath or Image object for the background

  • roi (tuple[int, int, int, int] | None) – rectangle of interest, on this operation it will act like a crop rectangle. The final image will be cropped by the roi values, the final image dimensions will be impacted by the roi values. roi tuple take four pixel values: (width1, width2, height1, height2)

  • force_fg_colorspace (str | None) – force foreground colorspace

  • force_bg_colorspace (str | None) – force background colorspace

Return type:
