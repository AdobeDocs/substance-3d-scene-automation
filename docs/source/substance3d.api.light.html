create()

Create a light in the given scene

Parameters:

  • light_type (str) – the type of light to create, must be a value in SUPPORTED_LIGHTS

  • parent (substance3d.api.core.stage_handler.StageHandler | substance3d.api.core.prim_handler.PrimHandler) – the parent under where the light will be created, can be scene or another object

  • identifier (str) – the name of the light

  • add_xform (bool) – Optional parameter to add this light to an xform

  • xform_name (str) – Optional parameter to name the added xform, only used if add_xform is true

  • properties (LightProperties) – Optional parameter to pass in light properties

Returns:

the created light object

Return type:
