init(scene, renderer_name='Eclair', implementation=substance3d.usd.hydra_renderer.RendererImplementation.CustomHydraEngineCpu, device=None, thread_limit=None)

Initialize the renderer and its context for the given scene.

Parameters:

  • scene (StageHandler) – The scene to render

  • renderer_name (str) – If given, the name of the renderer to use, otherwise the default one will be choose.

  • implementation (Any) – By default the hydra implementation is based on the pxr UsdImagingGLEngine. An experimental implementation HydraImplementation.SBS3D can be used with a custom hydra engine without opengl dependencies

  • device (str | None) – use enum values .class RenderingTechnique, only available for Eclair renderer. If set, override the scene’s RenderSettings value adobe.renderingTechnique.

  • thread_limit (int | None) – number limit of thread usage, default is 0 for no thread limit. If set, override the scene’s RenderSettings value adobe.threadLimit.

Return type:

StageHandler

render(scene, resolution=None, default_light=None, camera=None, background_image=None, background_resize=True, timecode=pxr.Usd.TimeCode.EarliestTime.GetValue, time_limit=0.0, samples=None, environment_background=False, passes=None, denoiser=False)

Render the given scene and store the rendering into the render buffer

Parameters:

  • scene (StageHandler) – The scene to render.

  • resolution (tuple | None) – The resolution of the rendering. If set, override the scene’s RenderSettings value resolution.

  • default_light (bool | None) – if True the renderer will use the default lights setup instead of the scene’s lights. If None, it will use the scene’s lights if they exist, and the default lights setup otherwise.

  • camera (substance3d.api.core.prim_handler.PrimHandler | None) – The camera to use for the rendering. If set, override the scene’s RenderSettings or RenderProduct camera if no camera is found a default one will be used by framing the scene.

  • background_image (str | substance3d.api.image.image.Image | pathlib.Path | None) – file path of a valid background image or a valid Image object.

  • background_resize (bool) – If True the given background image will be resized to the render resolution.

  • timecode (float | int) – the timecode to render the scene.

  • time_limit (float) – Time limit to stop rendering if the convergence takes longer than the given limit, value in seconds. If 0.0 or less the rendering will stop when it has converged.

  • samples (int | None) – Rendering quality. If set, override the scene’s RenderSettings value adobe.convergedPixelSamples.

  • environment_background (bool) – If True, display the environment dome light background. If set, override the scene’s RenderSettings value adobe.envLightCameraVisibility.

  • passes (list[substance3d.api.renderer.renderer_passes.Pass] | None) – A list of passes available in renderer.passes to render, if nothing if given the BEAUTY pass will be returned otherwise a dict of form {pass_name: image.Image}

  • denoiser (bool) – Enable denoiser. If set, override the scene’s RenderSettings value adobe.denoiserEnabled.

Return type:

substance3d.api.image.image.Image | dict[str, substance3d.api.image.image.Image]

set_groundplane(scene, enabled=True, reflections=False, reflections_opacity=1.0, reflections_roughness=0.0, shadows=True, shadows_opacity=1.0)
Set the groundplane feature for the renderer with the given arguments. If a groundplane already exist it will be

modified otherwise it will be created

Parameters:

  • scene (StageHandler) – The scene to render

  • enabled (bool) – Controls the enablement of the entire groundplane feature. False value will disable the feature completely

  • reflections (bool) – Controls the visibility of all reflections on the groundplane. False value will cause all reflections to be ignored

  • reflections_roughness (float) – When reflections are enabled, controls the material roughness of the reflections. Valid values are in the range 0-1

  • reflections_opacity (float) – When reflections are enabled, allows control of the opacity of the reflections. Valid values are in the range 0-1

  • shadows (bool) – Controls the visibility of shadows on the groundplane. False value will cause all shadows to be ignored.

  • shadows_opacity (float) – When shadows are enabled, allows control of the opacity of the shadows. Valid values are in the range 0-1.
