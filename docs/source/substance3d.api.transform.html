class Transformer

Bases: object

The Transformer class provides an object to easily transform a PrimHandler

Use Transformer’s properties and arithmetic assignment operations to transform the PrimHandler.

Example:

tr = Transform(my_cube)
# move the cube by adding values
tr.translate += (12, 0, 1)
# move the cube by subtract values
tr.translate -= (5, 2, 1)
# set by giving values
tr.translate = (0, 10, 0)
# same for rotation and scale
tr.scale += (0.2, 0.2, 0.2)
tr.rotate = (0, 90, 0)
