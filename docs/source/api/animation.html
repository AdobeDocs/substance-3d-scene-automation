substance3d.api.scene.animation

activate_timecode(parent, timecode)

Create a key context to add a keyed/sampled value for all prims transformed in the with block

Parameters:

  • parent (StageHandler) – a scene where the prim are animated

  • timecode (float | int | str) – the timecode where the key/sample will we created

  • label – optional label for this key to recall with animation.get_key(label)

Return type:

ActiveTimecode

create_timecode_label(parent, timecode, label)

Create a label for a timecode that can be used by other animation’s functions

Parameters:

  • parent (StageHandler) – a scene where the prim are animated

  • timecode (float | int) – the timecode attached to the label

  • label (str) – the label

Return type:

str

set_time_range(parent, range)

Set the start and end timecode for the given scene

Parameters:

  • parent (StageHandler) – a scene where the prim are animated

  • range (tuple[float, float]) – a tuple of two for start and end timecode
