create(primitive, parent, identifier='primitive')

Create a primitive object in the given scene or under the given object

Parameters:

  • primitive (str) – The primitive identifier GEOM_PRIMITIVES

  • parent (substance3d.api.core.stage_handler.StageHandler | substance3d.api.core.prim_handler.PrimHandler) – the parent under where the cube will be created, can be scene or another object

  • identifier (str) – the identifier name of the cube

Returns:

the created primitive object

cyclo(parent, identifier='cyclo', width=10.0, height=10, depth=10.0, round_radius=2.0, round_steps=30)
Create a cyclorama here is a mesh that includes a floor part and a wall part with a rounded transition,

which is often used as a backdrop in photography

Parameters:

  • parent (substance3d.api.core.stage_handler.StageHandler | substance3d.api.core.prim_handler.PrimHandler) – the parent under where the sphere will be created, can be scene or another object

  • identifier (str) – the identifier name of the sphere

  • width (float) – width size of the cyclo

  • height (float) – height size of the cyclo

  • depth (float) – the depth size of the cyclo

  • round_radius (float) – radius of the corner

  • round_steps (int) – step definitions of the corner

Returns:

the created cyclo

Return type:

PrimHandler
