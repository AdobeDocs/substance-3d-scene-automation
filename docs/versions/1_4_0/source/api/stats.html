substance3d.api.stats

collect_stats(entity, user_collectors=None, print_results=True, raise_errors=False)

Get stats for all the given argument, which can be a scene, an object or a list of objects

Parameters:

  • entity (substance3d.api.core.stage_handler.StageHandler | substance3d.api.core.prim_handler.PrimHandler | list[substance3d.api.core.prim_handler.PrimHandler]) – a scene, an object or a list of objects

  • user_collectors (list[substance3d.api.stats.context.BaseCollector]) – add user collector to get stats that not supported by default. Collector must be a class which inherits of stats.context.BaseCollector. See examples in stats.collectors.

  • print_results – if True a json dump will be printed

  • raise_errors – if True raise the errors encountered during stats collection

Returns:

the stats dict
