create_framed_cameras()

Create a set of cameras evenly spread out around selected objects

Parameters:

  • parent (substance3d.api.core.stage_handler.StageHandler | substance3d.api.core.prim_handler.PrimHandler) – the stage the camera belongs to

  • objects (list[substance3d.api.core.prim_handler.PrimHandler] | substance3d.api.core.prim_handler.PrimHandler | substance3d.api.core.stage_handler.StageHandler | None) – the object(s) the camera(s) should frame

  • number_of_cameras (int) – the number of cameras to create surrounding the object

  • identifier (str) – the name for the xform group all the cameras will be created in

  • aspect_ratio (float | None) – camera aspect ratio

  • timecode (float) – the timecode for the created camera

  • name (str) – the camera name before suffix incrementation

  • properties (CameraProperties) – optional parameter to pass in camera properties

  • orthographic (bool) – use orthographic projection

  • frame_fit (float) – 1.0 is a tight fit, increase to add space around the frame

Return type:

PrimHandler

:return group xform containing the cameras

Create a camera view oriented by a given vector. camera.ViewOrientations’s value can be used.

  • parent (substance3d.api.core.stage_handler.StageHandler | substance3d.api.core.prim_handler.PrimHandler) – The parent scene or object under which the camera will be created

  • identifier (str) – A valid identifier name for the created camera

  • orientation (tuple) – An orientation vector (x, y, z) or camera.ViewOrientations’s value

match_image_perspective(parent, image, objects=None, reset_longitude=False, frame_fit=1.5)

Match the given camera to the image perspective

Parameters:

  • parent (PrimHandler) – Camera to match with the perspective image

  • objects (list[substance3d.api.core.prim_handler.PrimHandler] | substance3d.api.core.prim_handler.PrimHandler | substance3d.api.core.stage_handler.StageHandler | tuple | None) – the list of objects or a bounding box tuple of (min, max) the camera should frame for the match

  • frame_fit (float) – 1.0 is a tight fit, increase to add space around the frame
