calculate_material_tiling_from_physicalsize(objects, physicalsize, physicalsize_unit=0.01, scene_unit=0.01)
From a mesh object and a material physical size, calculate the correct tiling to match the physical size according

to the object texel ratio. If several object are given an average texel ratio will be calculated.

Parameters:

  • objects (substance3d.api.core.prim_handler.PrimHandler | list[substance3d.api.core.prim_handler.PrimHandler]) – a mesh object

  • physicalsize (tuple[float, float, float]) – a physical size represented by a (x, y, z) vector

  • physicalsize_unit (float) – The physical size unit for this material in meter by unit, by default and for all Substance material the unit is cm (0.01)

  • scene_unit (float) – the scene unit in meter by unit, by default it’s cm (0.01)

Returns:

a tile tuple (x, y), z is not supported yet
