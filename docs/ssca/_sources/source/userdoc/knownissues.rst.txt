Known Issues
------------

Pixar and SSCA modules interaction
==================================

Pixar (``pxr``) and SSCA (``substance3d.api``) modules may interact with eachother and cause conflicts.
To ensure maximum compatibility between modules, ``substance3d.api`` ones should be loaded first.

Example:

.. code-block:: python

    from substance3d.api import scene, primitive, transform # ,...

    from pxr import Usd, UsdGeom, Gf, Sdf # ,...
