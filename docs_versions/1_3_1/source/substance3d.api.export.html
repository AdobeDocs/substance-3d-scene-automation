archive(parent, filename, bake_sbsar=True, flat=False, root_usda=False)

Create an usdz archive with a scene containing all its dependencies. Assets path will be modified to match the new location in the archive.

Parameters:

  • parent (substance3d.api.core.stage_handler.StageHandler | substance3d.api.core.prim_handler.PrimHandler) – Can be either a scene or an object to archive containing all its dependencies

  • filename (str | pathlib.Path) – filename of the archive file

  • bake_sbsar (bool) – If True all the sbsar assets will be baked as png files to avoid the need of the sbsar plugin If flat argument is True sbsar will be automatically baked

  • flat (bool) – If true the scene will be flattened before the archiving

  • root_usda – If True the root usd file in usdz archive will be saved as usda (instead of usd)

Returns:

the archive file path

Return type:

str
