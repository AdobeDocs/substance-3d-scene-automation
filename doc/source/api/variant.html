substance3d.api.variant

create(obj, variant_identifier, variant_set='variants')

Create a new Variant on this object into the given VariantSet (by default DEFAULT_VS_NAME). Enter with key word with in the variant edit context to create variation, the given object and all its children are include in the variation, if an object outside the given object hierarchy is edited it will not be taken into account in the variant, it will modify in a classic way.

Example:
scn = scene.load('simple_scene.usda')
a_cube = scn.objects['/root/cube']

# create a variant for cube into the default variant set "variants"
cube_var = variant.create(a_cube, 'red')
# enter in the variant edit context:
# all modification on the cube or its children will be applied in the variant
with cube_var:
    a_cube.primvars.displayColor = [(1.,0.,0.)]
Parameters:

  • obj (PrimHandler) – the object on which will be applied the variant, all its children are also included

  • variant_identifier (str) – the given identifier / name for this new variant

  • variant_set (str) – the variant set identifier in which the new variant will be created

Return type:

Variant

get(obj, variant_identifier, variant_set='variants')

Get the Variant of the given object corresponding to the given name. By default the Variant will be search into DEFAULT_VS_NAME variant set.

Parameters:

  • obj (PrimHandler) – the object on which is applied the variant

  • variant_identifier (str) – the given identifier / name for the desired variant

  • variant_set (str) – the variant set identifier in which the variant is created

Return type:

Variant

get_identifiers(obj, variant_set='variants')

Return a list of Variant’s identifier/name for the given VariantSet (by default DEFAULT_VS_NAME)

Parameters:

  • obj (PrimHandler) – the object on which is applied the variants

  • variant_set (str) – the variant set identifier in which the variants are created

Return type:

list[str]

get_selection(obj, variant_set='variants')

Return the identifier / name of the selected variant for the given VariantSet (by default DEFAULT_VS_NAME)

Parameters:

  • obj (PrimHandler) – the object on which is applied the variants

  • variant_set (str) – the variant set identifier in which the variants are created

Return type:

str

set(obj, variant_identifier, variant_set='variants')

  • variant_set (str) – the variant set identifier in which the variants are created
